# :gear: QuikBlog

Aplicação criada em React (CRA) usando NextUI (UI) e Stiches (css).

## :gear: Descrição
Aplicação para criação de posts simulando um blogs, com funcionalidades de curtir, visualizar e criar POST.

## Instalação

```bash
$ npm install

## Rodando a aplicação

```bash
# projeto
$ npm run start
```
## :gear: Techs utilizadas

- React JS
- Next UI
- Typescript
- Stitches
